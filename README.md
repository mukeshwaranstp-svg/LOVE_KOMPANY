# 💖 Love Kompany

A modern compatibility analyzer inspired by the classic FLAMES game.

Love Kompany is a fun web application that calculates compatibility between two names using a custom compatibility engine instead of the traditional FLAMES elimination algorithm. The project features a modern UI, smooth animations, responsive design, and an improved relationship hierarchy.

🌐 **Live Demo:** https://love-kompany.vercel.app

> ⚠️ This project is created for entertainment purposes only. The compatibility results are not scientifically accurate.

---

- 🎨 Modern Glassmorphism UI

## 💡 Why I Built This

This project wasn't just about building a fun website.

It was about exploring a new way of creating software with AI-assisted development.

A few years ago, I thought I needed to master every technology before I could build something meaningful. That often meant spending months learning HTML, CSS, JavaScript, React, and other tools before creating a complete application.

This project challenged that mindset.

Using AI-assisted development, I was able to transform an idea into a working product in a single night. But AI didn't replace my role. I still had to review the generated code, debug issues, redesign the compatibility algorithm, improve the user experience, and make technical decisions throughout the project.

For me, this project represents a new way of learning.

I believe learning programming is still essential. Understanding how software works allows you to improve, debug, and extend what AI generates. Ignoring AI, however, means missing one of the most powerful tools available to developers today.

My goal isn't to let AI build everything for me.

My goal is to become the kind of developer who knows how to combine programming knowledge with AI to build better software, faster.
